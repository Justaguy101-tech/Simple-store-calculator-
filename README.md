# A simple store calculator
This is a beginner-friendly way to under how Javascript operators and how the logic works.

---

## Features and how to use

- There are **four products** each with its corresponding price and you simply need to input how many you bought in the text box.
- Simple layout and easy to navigate
- Shows how much you spent in each product.
- It also has a **discount function** where if your total cost **exceeds 200** it discounts you with **20%**.

---

## How to use
- Simply input how many you bought at the text box and hit the total cost **Button**.
